**Itherum** es una micro-agencia de software argentina especializada en el desarrollo de productos digitales de alto impacto. Construimos tecnología con sentido común.

## Tech Stack

- **Frontend:** Next.js 16, React 19, TypeScript 5.7
- **Styling:** Tailwind CSS 4, CSS Variables (dark/light themes)
- **Animations:** Framer Motion 12
- **UI Components:** Radix UI, shadcn/ui
- **Forms:** React Hook Form + Zod
- **Deployment:** Vercel
- **Analytics:** Vercel Analytics

## Features

- Diseño premium con sistema de theming dual (dark/light)
- Animaciones fluidas con Framer Motion
- Componentes UI accesibles (Radix UI)
- Validación de formularios con Zod
- Totalmente responsive
- SEO optimizado
- Newsletter funcional
- cursor personalizado

## Setup Local

```bash
# Clonar el repositorio
git clone https://github.com/FacuPedrazzoli/itherum.git
cd itherum

# Instalar dependencias
pnpm install

# Iniciar desarrollo
pnpm dev
```

El proyecto estará disponible en `http://localhost:3000`.
